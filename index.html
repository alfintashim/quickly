Hi there!

Welcome to Quickly

Quickly adalah template dashboard CSS yang ringan dan responsif. Sangat cocok digunakan untuk keperluan projek dengan skala kecil. Quickly memiliki beberapa tema bawaan yang dapat diubah sesuka hati tanpa perlu konfigurasi javascript.

Download Now, Quickly!
Latest release is v0.1

Why So Quickly?

  • Lightweight

    Dibuat menggunakan CSS murni tanpa framework.

  • No Javascript

    Tak perlu konfigurasi javascript lebih lanjut

  • Responsive

    Ukuran sudah menyesuaikan dengan berbagai perangkat.

  • Custom Themes

    Memiliki tema yang dapat diganti dengan mudah.

Future Plan

Quickly masih dalam tahap pengembangan awal. Ada banyak sekali fitur yang ingin saya tambahkan di kemudian hari meliputi penulisan dokumentasi yang lengkap, kelas utility penunjang, kelas warna yang menyesuaikan tema, dukungan berbagai css framework, javscript API, dan berbagai tema baru lainnya, serta penyesuaian struktur agar template ini cocok digunakan sebagai dashboard maupun landing page.
